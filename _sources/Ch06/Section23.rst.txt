|Exercise Chunking 1| 
So it looks like the entire goal of that chunking process was to help us
figure out between these two operators which one we should actually
select. We decided that, if a fast ball is suggested, which it was, and
if two operators are suggested, which they were, and the batter is
Parra, which it is, then to dismiss the throw-fast-ball operator. That
means we only have one more operator suggested, throw-curve-ball. So
that would be selected.
